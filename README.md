# 🥪 Sandwich Calculator

A mobile-first Subway-style nutrition calculator.  
一個以手機體驗為核心的 Subway 營養計算工具  

---

## ✨ Features

- Protein efficiency sorting  
  依「蛋白質效率」排序（不是單純高蛋白）

- Real-time kcal & protein calculation  
  即時計算熱量與蛋白質

- Dynamic add-ons system  
  可自由新增／刪除加料

- Optional second sauce  
  支援第二種醬料（自動分攤熱量）

- iOS-like UX interactions  
  iOS 風格互動與動畫體驗  

---
