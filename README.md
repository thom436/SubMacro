# 🥪 Sandwich Calculator

A mobile-first Subway-style nutrition calculator.  
一個以手機體驗為核心的 Subway 營養計算工具。

---

## ✨ Features

- Modal-based flavor picker with grouping + kcal sorting  
  口味改為彈窗選單，依分類顯示並可直接比較熱量（kcal）。

- Add-on picker with grouped modal list  
  加料採彈窗分類清單，支援顯示 kcal、替換已選項目、防重複選擇。

- Drag-to-reorder add-ons  
  已選加料支援拖曳排序，並可快速刪除。

- Sauce modal picker + no-sauce option  
  醬料採彈窗選擇，支援第二種醬料與「不加醬」模式。

- Real-time kcal & protein calculation  
  即時計算熱量與蛋白質，並顯示明細與摘要。

- Mobile-focused interaction design  
  手機優先互動，包含滑動刪除、觸控友善操作與底部結果卡片。
